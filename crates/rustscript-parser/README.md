# rustscript-parser

Lexer and recursive descent parser for RustScript (`.rts`) source files.

Tokenizes and parses the full RustScript syntax — functions, classes, interfaces, generics, async/await, decorators, destructuring, and more — into a typed AST defined in `rustscript-syntax`.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
