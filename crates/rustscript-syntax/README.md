# rustscript-syntax

AST types, Rust IR, spans, and diagnostics for the RustScript compiler.

This crate defines the core data structures shared across the compilation pipeline: the RustScript AST (parsed from `.rts` source), the Rust IR (lowered output), source spans for error reporting, and the diagnostic infrastructure.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
