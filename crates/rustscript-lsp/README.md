# rustscript-lsp

Language server (LSP) for RustScript with diagnostics, hover, and completions.

Full LSP implementation with live diagnostics, type-aware hover, keyword and builtin completions, and a rust-analyzer proxy for go-to-definition and cross-file navigation. Works with VS Code and any LSP-compatible editor.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
