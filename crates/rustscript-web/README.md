# rustscript-web

WebAssembly bindings for client-side RustScript compilation.

Compiles RustScript to Rust entirely in the browser via WASM. Powers the [playground](https://rustscript.dev/playground) with live compilation, diagnostics, hover tooltips, and crate docs translation.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
