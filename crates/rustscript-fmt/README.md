# rustscript-fmt

Source code formatter for RustScript (`.rts`) files.

Opinionated, zero-config formatting. One style, no debates. Parses and re-prints `.rts` source with consistent indentation, spacing, and line breaks.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
