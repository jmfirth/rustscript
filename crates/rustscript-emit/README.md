# rustscript-emit

Rust IR to `.rs` source code emitter for the RustScript compiler.

Renders the lowered Rust IR into human-readable, idiomatic `.rs` files with source maps for error translation back to the original `.rts` source.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
