# rustscript

**Write TypeScript. Ship Rust.**

The RustScript compiler. Install with `cargo install rustscript`.

```bash
rustscript init my-app    # create a new project
rustscript build           # compile to native binary
rustscript run             # build and run
rustscript dev             # watch mode
rustscript fmt             # format .rts files
rustscript eject           # convert to pure Rust
```

`rsc` is available as a short alias.

See [rustscript.dev](https://rustscript.dev) for documentation, a live playground, and the crate browser. Source at [github.com/jmfirth/rustscript](https://github.com/jmfirth/rustscript).
