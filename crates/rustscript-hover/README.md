# rustscript-hover

Shared hover logic for the RustScript playground and LSP.

Provides TypeScript-grade hover information: function signatures, type annotations, doc comments, generic substitution, closure parameter inference, and 60+ builtin method descriptions. Used by both the WASM playground and the VS Code extension.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
