# rustscript-typeck

Type inference, resolution, and checking for the RustScript compiler.

Resolves type annotations, infers types from context, manages the type registry, and bridges RustScript types to their Rust equivalents.

Part of the [RustScript](https://rustscript.dev) compiler. See the [GitHub repository](https://github.com/jmfirth/rustscript) for the full project.
