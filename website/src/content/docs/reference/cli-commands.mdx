# CLI Commands

The `rsc` command-line tool compiles, runs, formats, and manages RustScript projects.

## rsc init

Create a new project:

```bash
rsc init my-project
```

Creates a directory with `rustscript.json`, a generated `Cargo.toml`, `.gitignore`, and `src/main.rts`.

### Templates

```bash
rsc init my-api --template web-server    # axum web server
rsc init my-tool --template cli          # CLI application
rsc init my-wasm --template wasm         # WebAssembly project
```

## rsc build

Compile to a native binary:

```bash
rsc build
```

This:
1. Parses `.rts` files
2. Type-checks
3. Lowers to Rust
4. Emits `.rs` files in-place next to source files
5. Merges detected dependencies into the project root `Cargo.toml`
6. Runs `cargo build`

The output binary is in `target/debug/`.

## rsc run

Build and execute in one step:

```bash
rsc run
```

Pass arguments to the compiled binary with `--`:

```bash
rsc run -- --port 8080 --verbose
```

Everything after `--` is forwarded to your program, not to `rsc`.

## rsc check

Type-check without building:

```bash
rsc check
```

Faster than `rsc build` because it skips code generation and `cargo build`. Use it for quick validation while editing.

## rsc fmt

Format `.rts` source files:

```bash
rsc fmt
```

Applies consistent formatting to all `.rts` files in the project. Analogous to `prettier` for TypeScript or `rustfmt` for Rust.

## rsc eject

Convert a RustScript project to pure Rust:

```bash
rsc eject
```

This:
1. Removes `rustscript.json` (the RustScript manifest)
2. Removes `/src/*.rs` from `.gitignore` so the generated Rust files are tracked
3. Leaves you with a standard Cargo project you can continue in pure Rust

Use this when you want to graduate a project out of RustScript without losing any code. The generated `.rs` files are idiomatic, human-readable Rust — they're a clean starting point.

## Common workflows

### Development loop

```bash
rsc run                    # Build and run
rsc check                  # Quick type check
rsc fmt                    # Format before committing
```

### Production build

```bash
rsc build                  # Compile to native binary
./target/debug/my-project  # Run the binary directly
```

## Exit codes

| Code | Meaning |
|------|---------|
| 0 | Success |
| 1 | Compilation error (parse, type check, or lowering) |
| Non-zero | `cargo build` or runtime failure |
