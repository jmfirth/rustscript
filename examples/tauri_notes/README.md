# Tauri Notes — RustScript Desktop App Example

A notes app demonstrating RustScript as the backend for a Tauri desktop application.

## Architecture

- **Backend**: RustScript (compiles to Rust via `rsc build`)
- **Frontend**: React + TypeScript
- **Shared types**: Generated by `rsc types -o frontend/src/types/`

## Type Safety Across the IPC Boundary

The backend defines types in RustScript:

```typescript
type Note = {
  id: u32,
  title: string,
  content: string,
  pinned: bool,
} derives Serialize, Deserialize
```

`rsc types` generates TypeScript definitions the frontend imports:

```typescript
import type { Note } from './types';
```

One source of truth. If you change the backend type, regenerate, and the frontend gets compile errors on any mismatch.

## Building

```bash
# Build the RustScript backend and generate frontend types
rsc build --emit-types frontend/src/types/

# Or generate types separately
rsc types -o frontend/src/types/

# Build the frontend
cd frontend && npm run build
```
